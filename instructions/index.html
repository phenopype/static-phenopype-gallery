Instructions#

Global instructions#

  1. Install phenopype (see installation instructions)

  2. Install jupyter: pip install jupyter notebook

  3. start the jupyter notebook server with jupyter notebook

Project specific instructions#

  1. Go to a project and read through the static copy of the jupyter notebook

  2. Download the materials (data, jupyter notebook, Python script, phenopype templates, etc.) and unpack the archive

  3. (optional) check if any extra libraries or packages (e.g., for phenopype plugins) need to be installed.

  4. Open the downloaded notebook from the notebook server (3.), and follow the instructions
